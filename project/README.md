# 画面の表示方法

## composer で module のインストール

```
composer install
```

## .env ファイルを作成し、 .env.example ファイルの中身をコピーする

## encryption key を設定する

```
php artisan key:generate 
```

## Laravel の起動

```
php artisan serve
```

## npm module のインストール

```
npm install
```

## FE 資材のビルド & 更新監視

```
npm run hot
```

## Storybook の起動

```
npm run storybook
```
